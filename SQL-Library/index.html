Soil Data Access Page

This SQL Soil Data Access Library page is an index to many examples of SQL queries that can be copied and pasted into the link below. You can then modify the query for different data fields and or parameters on the area.

Text highlighted in yellow can be changed for a particular area e.g. 'MO123' can be substituted for 'NE109' in the first query on the list; also you can use wildcards to get the entire state eg 'MO%' will extract the whole state of Missouri.
